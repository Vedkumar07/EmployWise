## Installation

1. Clone the repository
```bash
git clone https://github.com/Vedkumar07/EmployWise.git
cd EmployWise
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm run dev
```
